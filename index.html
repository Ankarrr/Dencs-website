About Dencs

Dencs is a Decentralized Elastic Network of Currency Services for payment system and remittance network by Dencs.org.

Dencs aims to enable safe, instant and low-cost global financial transactions and supports any credit system or other unit of value (internet bandwidth usage, mobile roaming data, utility usage tracking, etc.).

It is built upon a distributed, open-source protocol for currency services.
