Introduction

Dencs (pronounced like "Thanks") is a Decentralized Elastic Network of Currency Services for payment system and remittance network by DENCS.org. Dencs aims to enable safe, instant and low-cost global financial transactions and supports any credit system or other unit of value (internet bandwidth usage, mobile roaming data, utility usage tracking, etc.).

It is built upon a distributed, open-source protocol for currency services. At the core of Dencs, it is based around a shared, public, sequential transaction database, improved from Bitcoin blockchain. Let’s call it ledger.

Blockchain technology

The blockchain is simply a digital ledger. A ledger is the principal book or computer file for recording and totaling economic transactions measured in terms of a monetary unit of account by account type, with debits and credits in separate columns and a beginning monetary balance and ending monetary balance for each account.

The blockchain holds information about balances and transaction history of currencies and assets in all accounts. Participants in Dencs agree to the transactions added into the ledger via a mechanism of consensus algorithm which is reached in seconds.

Consensus algorithm

The fundamental consensus algorithm is elastic and secure, and ensures the trust between each participant. This mechanism utilized the amazing concept of proof of work, as a voting process to select a verifier once in a period. This verifier will check whether the transactions in that period are valid or not. All this allows for payments without the need for a centralized clearing house.
